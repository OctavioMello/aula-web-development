# 🌐 Estudos de Web Development

Repositório dedicado aos estudos e práticas realizadas durante as aulas de desenvolvimento web.

## 🚀 Objetivo
Consolidar conceitos fundamentais de desenvolvimento front-end através de exercícios práticos.

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 📌 Conteúdo
- Estrutura básica de páginas web
- Estilização com CSS
- Manipulação do DOM com JavaScript
- Interatividade com eventos

## 📚 Aprendizados
- Criação de interfaces simples
- Integração entre HTML, CSS e JavaScript
- Organização de código para projetos web

## 📂 Estrutura
- /html → arquivos de estrutura
- /css → estilos
- /js → scripts

## ⚠️ Observação
Este repositório faz parte do meu processo de aprendizado e evolução na área de desenvolvimento web.
